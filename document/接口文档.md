返回格式：
```json
[
  {"title":"xxx","author":"xxx","content":"xxx","mean":"xxx","kind":"xxx"},
  {"title":"xxx","author":"xxx","content":"xxx","mean":"xxx","kind":"xxx"},
  {"title":"xxx","author":"xxx","content":"xxx","mean":"xxx","kind":"xxx"}
]
```

## 根据种类获取
请求格式：`/poem/kind(POST)`
```json
{
   "kind" : "一年级上册"
}
```

返回格式：
```json
[
  {"title":"xxx","author":"xxx","content":"xxx","mean":"xxx","kind":"xxx"},
  {"title":"xxx","author":"xxx","content":"xxx","mean":"xxx","kind":"xxx"},
```
